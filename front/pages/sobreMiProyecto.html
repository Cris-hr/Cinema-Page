PAGINA WEB DE PELICULAS

I. Resumen del proyecto: 

El objetivo de aprendizaje de este proyecto es diseñar y construir una interfaz web donde se pueda visualizar y manipular data,
entendiendo primeramente las necesidades del usuario. El producto final será una página web que permite visualizar data de películas
y una pequeña descripción.

II. Diseño de la interfaz de usuario: 

El prototipo de alta fidelidad se desarrolló en Figma, teneniendo en cuenta que la interfaz sea amigable,facil, colores armoniosos e intuitiva
para el usuario final.

III. Consideraciones Técnicas: 

La lógica del proyecto la he implementado completamente en JavaScript (ES6), HTML, CSS y Bootstrap como frameworkn para estilar algunas 
paginas.

IV. Desarrollo del proyecto: 

El proyecto inicia en una página principal que contiene una nav con enlaces a diferentes páginas que muestran respectiva informacion
(Sobre mi proyecto, Historia del cine y Webs recomendadas). Dentro del cuerpo de la página se pude visualizar una sección de películas.
